# 游戏重大更新实机演示 (Walkthrough)

在本次更新中，我们为《主神空间》的视觉引擎和核心机制完成了四项重大的更新迭代，并修复了已知的界面卡死 Bug。另外，我们还扩充了装备词条系统，新增了装备盲盒抽取功能。

## 1. 视觉化战斗进化 (技能与特效 VFX)
我们引入了结构化的方式，在回合制战斗中渲染特效和技能表现。
- 在 `gui/combat_canvas.py` 中新增了 `ParticleSystem` 和特效发射队列。
- 重构了 `combat_calc.py` 中的 `use_skill` 函数，现在会查找技能特征及颜色，并直接发射视觉特效，而不再仅仅是打印文字。
- 实现了伤害结算延迟，使其与投射物击中的动画表现同步。

## 2. 团队战斗系统
现在的战斗不再局限于 1v1 单挑。
- 更新了 `combat_canvas.py`，支持在左侧绘制多个 `player_team`（玩家小队）角色，右侧绘制多个 `enemy_team`（敌人）角色。
- 修改了 `combat_calc.py`，为每个战斗参与者分配了唯一的 `actor_id`。
- 实现了基于 `agi`/`speed`（敏捷/速度）优先级的真实回合队列，自动遍历所有存活的战斗者。

## 3. 视觉化强化大厅 (技能树 UI)
原先纯文本的“强化大厅”菜单，现已被彻底改造为一个完全可交互的视觉技能树。
- 创建了 `gui/enhancement_hub.py`，具有节点拖拽和画布平移逻辑。
- 节点类别包含：基础属性提升、主动/被动技能、血统进化。
- 用户界面能够根据前置条件自动计算技能解锁情况，并在玩家点击节点进行购买时，实时更新玩家的 JSON 数据面板。

## 4. 2D 网格地图任务探索
为“任务世界”设计了全新的移动和交互方式。
- 将 `world_engine.py` 中的随机文本选择探索，替换为了 2D 瓷砖阵列生成器。
- 开发了 `gui/map_renderer.py` 用于渲染网格地图、绘制战争迷雾，并为不同类型的房间赋予颜色代码（敌人、宝箱、事件、首领 Boss）。
- 通过点击相邻的格子可触发移动事件并遭遇不同类型的房间，全过程完全在视窗画布中实时呈现。

## 5. 装备词条大扩充与盲盒抽卡系统
为了解决新手发育问题，并丰富游戏的装备多样性：
- 我们在 `utils/equipment_gen.py` 中增加了几十种前后缀和核心物品类别，使得随机装备生成器的词条池扩充到了 **上万种以上的组合可能**。
- 在 `main_god_space.py` 的兑换商城中新增了 **“🎲 随机装备盲盒”** 功能。
- 玩家现在可以花费积分去抽取与其当前等级匹配的完全随机属性的定制装备（分普通盲盒与高级盲盒，有着不同的保底机制），极大地丰富了获得极品神装的乐趣。

### 测试记录
所有更新内容均已通过 `gui_main.py` 路由测试。Tkinter 窗口已经能够根据游戏的状态上下文，顺畅地在中央文本输出区、战斗渲染器、强化大厅渲染器以及 2D 图形探索地图之间无缝切换（我们刚才也修复了由于引擎调用错误导致进入任务界面卡死的 Bug！）。
您可直接运行全新打包好的 `MainGodSpace_GUI.exe` 进行沉浸式体验！

## 6. 超越级海量扩展内容 (The Grand Expansion)
在您赋予的超级权限下，游戏底层世界被极大拓展：
- **万界世界观**：新增修真界、克苏鲁神话体系、侏罗纪原始恐龙世界、机械飞升、甚至是魂系（黑暗之魂）等 10+ 种各具特色的新位面。
- **技能库爆增**：主动与被动技能（包含大火球术、暗影步、自在极意等）扩容至近 30 种完整打通逻辑流的组合。
- **神级商店**：主神商城全面进货！包括超强保底**盲盒抽奖**，以及上古盘古斧、誓约胜利之剑等传说装备，还有新增了赛亚人、氪星人、机械飞升等顶尖血统。
- **地图奇遇事件**：地图中曾经单一的事件，变成了包含了远古神龛治愈、流浪黑市商人、甚至是血祭诅咒祭坛等丰富多元的抉择。

新手不再坐牢！在创建新轮回者角色的捏人环节之后，主神将自动倾泻：
- **3000 点初始高级积分**，让你刚出生即富有。
- **保底稀有以上的传说/史诗装备盲盒开箱**，自带前缀强化效果。
